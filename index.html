2015年总结

转眼，一年又过去了，每天上班地铁路上的一小时可以收获很多，列一下今年的读书清单

  1. 《大型分布式网站架构设计与实践》
  2. 《大型网站系统与Java中间件实践》
  3. 《深入分析Java Web》
  4. 《大规模分布式系统架构与设计实战》
  5. 《Java加密与解密的艺术》

如果有两个delete
kid1 与 kid2是索引字段
语句1 delete from table where kid1=1 and kid2=2;

语句2 delete from table where kid1=1 and kid2=3;

这样的两个delete 是不会锁表的

语句1 delete from table where kid1=1 and kid2=2;

语句2 delete from table where kid1=1 ;

这样的两个delete 会锁表


总结：同一个表，如果进行删除操作时，尽量让删除条件统一，否则会相互影响造成锁表
